# Macintosh 復古風格履歷網站

這是一個個人履歷網站，其設計靈感來自於經典的 Macintosh 復古作業系統介面。

本專案使用純 HTML 和 CSS 打造，模擬了多視窗、彈出視窗和底部 Dock 欄的視覺與互動體驗。

## 🚀 功能特色

* **復古視覺**：模仿經典 Mac OS 介面，包含視窗標題欄、關閉按鈕和像素字體 (`DotGothic16`)。
* **純 CSS 互動**：全站**不使用任何 JavaScript**。所有的互動，包括：
    * 透過 Dock 欄「開啟」與「關閉」視窗。
    * 點擊視窗彈出「Modal 彈窗」。
    * ...均使用 HTML 的 `<input type="checkbox">` 和 `<label>` 技巧達成。
* **動態圖表**：在「程式能力」彈窗中，使用純 CSS 動畫實現的動態圓餅圖和直條圖。
* **完整響應式設計 (RWD)**：針對三種主要的瀏覽情境提供完全不同的佈局。

## 🖥️ 響應式佈局 (RWD)

本網站針對不同裝置提供了三種獨特的佈局：

1.  **電腦版 (Desktop > 1024px):**
    * 完整的桌面體驗，所有視窗均為絕對定位，模擬真實作業系統的隨機擺放感。

2.  **橫向平板 (Landscape Tablet <= 1024px):**
    * 客製化的網格佈局，內容依序排列為：
        * 2x2 網格（個人、技能、攝影、聯絡）
        * 3x1 垂直堆疊（專案 1, 2, 3）
        * 置中的版權視窗

3.  **直向平板/手機 (Portrait <= 1024px):**
    * 單欄垂直堆疊佈局，將所有視窗轉換為易於滾動閱讀的卡片。

## 🖱️ 網站操作指南 (純 CSS 互動)

本網站旨在模擬經典 Mac OS 的視窗化操作，但所有功能皆透過純 HTML 與 CSS 實現。

### 1. 視窗的開啟與關閉 (Dock 欄)

所有的視窗（App 或 Folder）預設為開啟。您可以使用最下方的 **Dock 欄**來控制視窗的顯示與隱藏：

* **開啟/關閉**：點擊 Dock 欄中對應的圖標 (如 **About**, **Skills**, **Photo** 等)，即可在桌面上隱藏或重新顯示該視窗。

### 2. 應用程式彈出視窗 (Modal)

您的主要資訊區塊點擊後會彈出一個全螢幕的**模態視窗 (Modal)**，用於展示詳細內容、圖表或多媒體素材。

* **開啟彈窗**：
    1.  點擊桌面上的**任何一個資訊視窗**的內容區域（例如，點擊 "About_Me.txt" 視窗、"Skills.app" 視窗，或任一專案視窗）。
* **關閉彈窗**：
    1.  點擊彈窗右上角的 **[X]** 關閉按鈕。
    2.  點擊彈窗外部的**半透明黑色遮罩區域**。

### 3. 技能與多媒體展示

* **程式能力 (Skills.app)**：開啟彈出視窗後，您將看到使用純 CSS 動畫實現的圓餅圖和直條圖，以及詳細的技能說明。
* **攝影經歷與專案**：彈出視窗會展示更大的圖片或內嵌影片。

### 4. 響應式佈局轉換 (RWD)

* **電腦/大平板**：視窗為自由浮動的層疊模式。
* **橫向平板 (Landscape Tablet)**：視窗將自動切換為**簡潔的 2x2 網格佈局**（只顯示個人資訊、技能、攝影和聯絡我），以優化空間利用。
* **直向平板/手機 (Portrait)**：視窗會自動轉換為**單欄垂直堆疊**的模式，確保在移動設備上的易讀性。
